**Interactive visualization of GitHub repositories for easy exploration of file structures.**

---

## Overview

This project allows users to **visually explore the structure of any public GitHub repository** in an intuitive and interactive way. It represents repositories as a **radial tree**, where directories and files are nodes. It features a robust **Spring Boot backend** that handles API requests, caches history in **MongoDB**, and is integrated with **OpenAI** for future code analysis features.

- **Radial Tree Layout:** View your repositoryâ€™s folder structure in a compact, circular layout.
- **Interactive Previews:** Hover over files to see a quick preview and click to view the full content.
- **Search & Highlight:** Quickly locate files and directories with a live search.
- **Zoom & Pan:** Navigate large repositories easily with zooming and panning.

## Usage

1. **Enter a GitHub repository URL** (e.g., `https://github.com/facebook/react`).
2. **Visualize the folder structure** generated by the D3.js engine.
3. **Hover over a file node** to see a preview.
4. **Click on a file node** to expand the full file content (fetched securely via the backend).
5. **Search** for specific files to highlight them in the tree.

---

## How It Works

**Flow:**
1. **User Input:** User enters a GitHub URL in the React frontend.
2. **Backend Request:** The frontend sends a request to the **Spring Boot** backend.
3. **Data Fetching:**
   - The backend parses the URL and fetches the repository tree using the **GitHub REST API**.
   - The request details are logged into **MongoDB** for history tracking.
4. **Visualization:** The structured data is sent back to the frontend, where **D3.js** constructs the radial tree.
5. **Content Retrieval:** When a user clicks a file, the backend securely proxies the content from GitHub to avoid CORS issues.

## Setup & Installation

### Prerequisites
- Node.js & npm
- Java 17+ & Maven
- MongoDB (Local or Atlas connection string)

### Backend (Spring Boot)
1. Navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Configure `src/main/resources/application.properties`:
   ```properties
   spring.data.mongodb.uri=YOUR_MONGODB_URI
   github.api.token=YOUR_GITHUB_TOKEN (Optional)
   openai.api.key=YOUR_OPENAI_KEY (Optional)
   ```
3. Run the application:
   ```bash
   ./mvnw spring-boot:run
   ```
   The server will start on `http://localhost:5000`.

### Frontend (React)
1. Navigate to the frontend folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:5173`.
